# Team3 Checkmate Wiki

Checkmate 프로젝트의 협업 규칙, 기술 문서, 개발 기록을 한곳에서 관리하는 위키입니다.

## 🗂️ 문서 구조

- `🤝 협업`: 팀 규칙/컨벤션 문서
- `⭐️ 프로젝트`: 아키텍처, 기술 선택 배경
- `📖 Checkmate Wiki`: 개인 학습/공유 문서
- `🛠️ 개발 히스토리`: 기능 설계/의사결정 기록 (간단한 변경도 문서화)
- `📅 데일리스크럼`: 일일 진행 상황
- `📝 회고`: 주간 회고 및 개선 포인트
